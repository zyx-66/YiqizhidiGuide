# 遗弃之地 · 攻略助手

> 中式民俗恐怖 · 横版策略塔防 Roguelike 手游攻略

## 🎮 关于游戏

《遗弃之地》是一款中式民俗恐怖题材的横版策略塔防手游，融合了 Roguelike 随机性元素。采用黑白剪影 + 中式皮影的美术风格，玩家扮演「修法者」，通过符咒合成、手牌管理和天书养成来对抗敌人。

## ✨ 攻略助手功能

- 🏠 **首页大厅** — 天书强度天梯、新手避坑指南、快速导航
- 🗺️ **关卡攻略** — 25关主线全配置 + 特殊关卡（黑雾之境、世界BOSS）
- 📖 **图鉴大全** — 天书 / 鬼仆 / 装备 / 符咒 / 资源 五大图鉴（支持搜索和筛选）
- 💡 **策略宝典** — 四大流派推荐、五帝钱优先级、商店购买指南、养成路线图
- 🔍 **全局搜索** — 支持 `Ctrl+K` 快捷键，跨模块搜索

## 🚀 使用方法

直接双击 `index.html` 即可在浏览器中打开使用，无需任何服务器。
